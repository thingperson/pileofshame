# Inventory Full: 50 Theme Ideas

> Ranked 1-50. #1 = Brady's top pick. Dual-scored for **Brady appeal** (bold, unexpected, personality) and **user appeal** (recognizable, distinct, broadly liked).
>
> Existing themes: Dark, Light, 90s, 80s, Future, Dino, Weird, ULTRA, Void, Cozy, Minimal, Tropical, Campfire

---

## Existing Theme Notes: Merge/Overlap Audit

| Theme A | Theme B | Concern | Recommendation |
|---------|---------|---------|----------------|
| Void | Minimal | Both strip UI down, dark + sparse | Keep both — Void is emotional/dramatic, Minimal is rational/Swiss. Different audiences. |
| 80s | ULTRA | Both neon-heavy, loud, retro-maximalist | Monitor. If users only pick one, consider merging ULTRA into 80s as a toggle. |
| Dark | Void | Dark is default, Void is dark-stripped | Fine — Void is a deliberate subtractive experience. Dark is just... dark mode. |
| Cozy | Campfire | Both warm-toned, comfort-oriented | Fine for now. Campfire is more outdoorsy/amber, Cozy is more pastel/indoor. Diverge the palettes further. |
| Future | 80s | Both use neon, both "retro-future" adjacent | Future should lean clean/minimal sci-fi (Interstellar), 80s should lean dirty/VHS (Miami Vice). |

## The 50

### 1. Glitch
- **Emoji**: `⚡`
- **Vibe**: Corrupted data, broken rendering, beautiful errors. The screen is lying to you.
- **Palette**: Black base, hot magenta (#FF0066), electric cyan (#00FFFF), occasional white artifacts. RGB channel-split accents.
- **Feasibility**: Medium — needs CSS glitch animations (text displacement, color channel offsets, scanline overlays). No custom art.
- **Why #1**: This is pure Brady energy. It's weird, it's loud, it's gaming-native, and nobody else has it.

### 2. Liminal Space
- **Emoji**: `🚪`
- **Vibe**: Backrooms energy. Empty malls at 3am. Something is wrong but you can't name it.
- **Palette**: Sickly fluorescent yellow (#F5E6A3), institutional beige (#D4C9A8), washed-out teal (#7BA7A7), off-white (#F0EDE5). Flat, unsettling.
- **Feasibility**: Medium — needs subtle ambient CSS animations (flickering light, very slow scroll parallax). A repeating ceiling tile pattern bg would be ideal.
- **Why**: Genuinely unsettling in a fun way. Internet-native aesthetic. Nobody would expect a backlog app to do this.

### 3. Neon Arcade
- **Emoji**: `🕹️`
- **Vibe**: Standing in a dark arcade, bathed in cabinet glow. Insert coin.
- **Palette**: Near-black (#0A0A12), hot pink (#FF2D95), electric blue (#00D4FF), acid green (#39FF14), warm amber (#FFB627) for highlights.
- **Feasibility**: Easy — CSS glow effects (text-shadow, box-shadow), dark background. Already partially described in parking lot.
- **Why**: Universal gaming nostalgia. Instantly recognizable. High "show your friends" factor.

### 4. Terminal
- **Emoji**: `>_`
- **Vibe**: Green phosphor CRT. You're hacking your own backlog. Command line aesthetic.
- **Palette**: Pure black (#000000), phosphor green (#33FF33), dim green (#1A6B1A), amber variant option (#FFB000). Monospace everything.
- **Feasibility**: Easy — CSS only. Monospace font, scanline overlay (repeating-linear-gradient), cursor blink animation, text-shadow glow.
- **Why**: Dev-brained gamers will lose their minds. Perfect intersection of gaming + tech culture.

### 5. Fever Dream
- **Emoji**: `🫠`
- **Vibe**: Nothing makes sense and you're fine with it. Oversaturated, slightly wrong, beautiful chaos.
- **Palette**: Shifting gradients — deep purple (#6B0F6B) to hot coral (#FF6B6B) to gold (#FFD700). Colors that shouldn't coexist. Background subtly shifts on scroll.
- **Feasibility**: Medium — CSS gradient animations, hue-rotate on hover, slightly rotated cards. No custom assets but needs careful animation work.
- **Why**: This is the theme you show someone to prove your app has personality. Conversation starter.

### 6. Cardboard / Indie Zine
- **Emoji**: `📦`
- **Vibe**: Kraft paper, tape, marker scrawl. Your backlog as a hand-assembled zine from a convention floor.
- **Palette**: Kraft brown (#C4A77D), cream (#FFF8E7), black marker (#1A1A1A), red marker (#D94040), blue ballpoint (#3355AA).
- **Feasibility**: Medium — needs a paper texture background (CSS or small repeating image), rough/hand-drawn border style. Consider a handwriting-adjacent font for headers.
- **Why**: Tactile, warm, completely different from every other gaming app. Indie game energy.

### 7. Vaporwave
- **Emoji**: `🏛️`
- **Vibe**: Dead malls, Windows 95, Greek statues, smooth jazz. Nostalgia for a future that never happened.
- **Palette**: Pastel pink (#FFB8D0), soft cyan (#80FFEC), lavender (#C4B0FF), warm peach (#FFCBA4), deep purple background (#2D1B69).
- **Feasibility**: Easy — CSS gradients, maybe a subtle grid-floor pattern (linear-gradient). Vaporwave is mostly color + typography.
- **Why**: Peak internet aesthetics. Huge built-in audience. Visually stunning.

### 8. Brutalist
- **Emoji**: `🏗️`
- **Vibe**: Raw concrete. Exposed structure. Anti-design that IS the design.
- **Palette**: Concrete grey (#B0B0B0), near-black (#1A1A1A), raw white (#FFFFFF), industrial yellow (#FFD600) for warnings/accents. No gradients anywhere.
- **Feasibility**: Easy — CSS only. Thick borders, system fonts, harsh alignment, all-caps headers. Less styling = more brutalist.
- **Why**: Genuinely polarizing — people who love it LOVE it. Great counterpoint to Cozy.

### 9. Deep Ocean

- **Why**: Stunning dark theme variant that isn't just "dark mode." Unique.

### 10. Library Card
